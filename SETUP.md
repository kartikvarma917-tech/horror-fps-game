# Installation and Setup Instructions for Horror-FPS-Game

## Prerequisites
- Make sure you have [Unity](https://unity.com/) installed (version 2020 or later).
- Download and install [Visual Studio](https://visualstudio.microsoft.com/) for script editing.

## Installation Steps
1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/horror-fps-game.git
   cd horror-fps-game
   ```

2. **Open the project in Unity**:
   - Launch Unity Hub and open the cloned project folder.

3. **Import required packages**:
   - Go to `Window` > `Package Manager` in Unity and install any required packages mentioned in the project documentation.

4. **Configure project settings**:
   - Go to `Edit` > `Project Settings` and configure settings based on your requirements.
   - For VR support, ensure that the XR settings are properly configured.

5. **Build the project**:
   - Once your changes are made and gameplay is ready, you can build your project by going to `File` > `Build Settings`.
   - Choose your target platform and click `Build`.

## Running the Game
- After building, you can run the game by executing the generated build files.

## Troubleshooting
If you encounter issues during setup, check the following:
- Ensure all prerequisites are installed correctly.
- Refer to the [Unity documentation](https://docs.unity3d.com/) for specific error messages or issues.
